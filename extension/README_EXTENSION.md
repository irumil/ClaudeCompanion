# ClaudeCompanion Firefox Extension

Браузерное расширение для передачи cookies и URL на локальный сервер десктопного приложения.

## Установка иконок

Перед упаковкой расширения необходимо добавить иконки:
- `icon48.png` - 48x48 пикселей
- `icon96.png` - 96x96 пикселей

Вы можете создать простые иконки с логотипом Claude или любым другим изображением.

## Упаковка расширения

1. Откройте Firefox
2. Перейдите в `about:debugging#/runtime/this-firefox`
3. Нажмите "Load Temporary Add-on"
4. Выберите файл `manifest.json` из папки `extension`

Для постоянной установки:

1. Создайте ZIP архив из содержимого папки `extension`
2. Переименуйте `.zip` в `.xpi`
3. Подпишите через AMO (self-distribute)
4. Установите подписанный `.xpi` файл

## Настройки

После установки:
1. Кликните правой кнопкой на иконке расширения
2. Выберите "Настройки"
3. Укажите порт (по умолчанию 8383)
4. Проверьте соединение

## Использование

Расширение автоматически срабатывает при посещении страниц вида:
`https://claude.ai/*/organization/{UUID}/usage`

При обнаружении такой страницы расширение:
1. Собирает все cookies для домена `.claude.ai`
2. Отправляет их вместе с URL на локальный сервер
3. Десктопное приложение использует эти данные для API запросов
